Map of Diseases

          In the end, we aimed to produce a map, and this was the hardest part of our project. The map above is an interactive map with points marking the recorded locations of smallpox during a particular outbreak, with the date viewable from each point. While currently this website only contains a screenshot of the map, we will soon be integrating the actual code with the website so readers can interact with the information. We expected to do more with our map, but due to the complications of learning a new coding language and mapping tools, our first map took longer than we anticipated. As the project continues, we will add more elements to the map to better demonstrate and accentuate our findings. Our eventual map will demonstrate the spread of both syphilis and smallpox, with different overlays including a map of European colonies and one of Native American tribes, so we can place these dates and locations in context. The other significant feature we still haven’t added is the timeline progression within the map, so that we can view different rates of spread.

          The process for creating the map began first with data collection. Because there is almost no digital data on this subject, we had to search multiple books for information on the dates and locations of outbreaks. This took a significant amount of time, as we were manually collecting information. We still need to collect further information to add more evidence supporting our claims, which will be a continued process of reading historical texts.

          We took the data we found and placed it in a spreadsheet and exported the data, using code that formatted the locations names and added their latitude and longitude. Then we took the formatted data, and placed it in a program called GeoJSON, which took the information in the spreadsheet and mapped it with the date located at each location point. This is the point at which the screenshot above was taken, with the map and GeoJSON code. This process took more work than expected, as the code originally written did not distinguish between East/West and North/South in the coordinates, so the first produced map originally mapped every point to India. With further adjustments, we managed to get each of the data points in the right location.

          Next, we will export this code as a GeoJSON file, and implement further Python code to set up an HTML web server to host our map. From there, we can then put the interactive map on the website, with the extra features mentioned previously. This process is difficult as the website is hosted on one account, while the map is hosted in a different location, so coordinating these and implementing them simultaneously will take further work.
